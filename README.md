## Getting Started

open Ai에서 APi Key 발급 이후 프로젝트 클래스패스에 apikey.yml 파일 생성 후 키 입력

### Prerequisites
* apikey.yml Project src/main/resources/ (ClassPath) 파일 생성
  ```sh
  openai:
    apikey : your api key
